## 💻 Sobre o Projeto

O **Tecnologia sem Mistérios** nasceu com o propósito de democratizar o acesso à informação e promover a inclusão digital. Sabemos que muitas pessoas, especialmente idosos ou indivíduos que não cresceram na era digital, enfrentam barreiras e ansiedades ao interagir com smartphones e computadores.

Esta plataforma funciona como um guia interativo, simplificando conceitos complexos e ensinando o uso de ferramentas cotidianas de forma paciente, visual e didática.

> 🌐 **Acesse a Aplicação:** [Clique aqui para abrir o site público](https://cryslemos.github.io/Tecnologia-sem-Misterios/)](https://cryslemos.github.io/Tecnologia-sem-Misterios/)


## 🛠️ Tecnologias Utilizadas

A aplicação foi construída visando leveza, rapidez e compatibilidade com múltiplos dispositivos:

* **Front-end:** HTML5 semântico e CSS3 responsivo.
* **Comportamento:** JavaScript assíncrono para interações e gerenciamento de estados.
* **Arquitetura Mobile:** Conceitos de Mobile-First para garantir que a experiência no celular seja idêntica à de um aplicativo nativo.

---

## 🚀 Características como PWA (Progressive Web App)

Graças à implementação de tecnologias modernas de web app, este projeto oferece:

* **Instalabilidade:** O usuário pode "Adicionar à tela de início" no celular, gerando um ícone no menu de aplicativos sem precisar passar pela Play Store ou App Store.
* **Service Workers (`sw.js`):** Configuração estruturada para estratégias de cache, melhorando o carregamento da página.
* **Manifest (`manifest.json`):** Definição de cores temáticas, ícones adaptativos e modo de exibição *standalone* (tela cheia, ocultando a barra do navegador para parecer um app).

## ⚙️ Funcionalidades Planejadas / Disponíveis

* **Tutoriais Passo a Passo:** Guias interativos e simplificados com foco em usabilidade.
* **Interface de Alta Legibilidade:** Fontes maiores, botões espaçados e alto contraste para facilitar a leitura.
* **Navegação Intuitiva:** Redução de menus complexos para evitar a sobrecarga cognitiva do usuário.
